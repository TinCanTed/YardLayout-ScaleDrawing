ScaleDrawing – Install Instructions for Kerri

1. Download this ZIP from GitHub Releases.
2. Extract it to a folder (right-click → Extract All).
3. Double-click setup_gui_scale_drawing_1.0.0.exe
4. Follow the installer wizard.
5. The program will appear in your Start Menu.

If Windows SmartScreen warns you:
Click "More info" → "Run anyway".
